# Web版日签生成器

## 项目简介

本项目是一个简洁易用的网页版日签生成器，用户可以自定义输入内容、上传头像和插画，自动生成带有日期、农历和星期信息的精美日签图片，并支持一键导出分享。

---

## 功能规划

1. **自动生成日期、农历、星期等信息**
2. **用户自定义输入内容**
   - 昵称
   - 签名
   - 个性化内容（金句、心情等）
3. **用户上传头像和插画（可选）**
4. **实时预览日签效果**
5. **一键生成并导出日签图片（PNG格式）**

---

## 技术选型

- 前端框架：React（推荐）或纯 HTML+JS
- 日期与农历：JS 日期库 + 农历库（如 `chinese-lunar` 或 API）
- 图片合成与导出：HTML2Canvas
- 文件上传与本地预览：input[type="file"]

---

## 开发计划

1. 初始化项目结构
2. 实现输入区（昵称、签名、自定义内容、头像、插画上传）

---

## 使用说明

1. 输入个人信息和自定义内容
2. 上传头像和插画（可选）
3. 预览生成的日签
4. 点击“生成日签图片”按钮，保存图片到本地
